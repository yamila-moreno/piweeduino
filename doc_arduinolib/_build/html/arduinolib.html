arduinolib.arduinolib module¶

A library to interface Arduino through serial connection

Developed for the piweek, dec 2013

Bases: builtins.object

Models an Arduino connection

analog_read(pin_number)[source]¶

Performs an analog read on pin_number and returns the value (0 to 1023) Internally sends b’RA{pin_number}’ over the serial connection

analog_write(pin_number, analog_value)[source]¶

Writes the analog value (0 to 255) on pin_number Internally sends b’WA{pin_number}:{analog_value}’ over the serial connection

digital_read(pin_number)[source]¶

Performs a digital read on pin_number and returns the value (1 or 0) Internally sends b’RD{pin_number}’ over the serial connection

digital_write(pin_number, digital_value)[source]¶

Writes the digital_value on pin_number Internally sends b’WD{pin_number}:{digital_value}’ over the serial connection

set_pin_mode(pin_number, mode)[source]¶

Performs a pinMode() operation on pin_number Internally sends b’M{mode}{pin_number} where mode could be: - I for INPUT - O for OUTPUT - P for INPUT_PULLUP
